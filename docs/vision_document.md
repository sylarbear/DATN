# TÀI LIỆU HƯỚNG DẪN SỬ DỤNG (Vision Document)
## Hệ thống English Learning - Nền tảng học tiếng Anh trực tuyến

---

## 1. Giới thiệu

### 1.1 Mục đích
English Learning là hệ thống website học tiếng Anh trực tuyến theo chủ đề, tích hợp trí tuệ nhân tạo (AI) đánh giá kỹ năng phát âm. Hệ thống cho phép người dùng học từ vựng, làm bài kiểm tra, luyện nói và theo dõi tiến độ học tập.

### 1.2 Đối tượng sử dụng
- **Học viên (Student):** Người muốn học và luyện tiếng Anh
- **Quản trị viên (Admin):** Người quản lý hệ thống, nội dung và người dùng

### 1.3 Yêu cầu hệ thống
- **Trình duyệt:** Chrome / Firefox / Edge (phiên bản mới nhất)
- **Kết nối:** Internet (để sử dụng AI và thanh toán)
- **Thiết bị:** Máy tính, laptop, tablet (có hỗ trợ responsive)
- **Microphone:** Cần có để sử dụng tính năng luyện nói

## 2. Hướng dẫn cho Học viên

### 2.1 Đăng ký tài khoản
1. Truy cập trang web → nhấn **"Đăng ký"**
2. Điền thông tin: Tên đăng nhập, Email, Họ tên, Mật khẩu
3. Nhấn **"Đăng ký"** → Hệ thống chuyển đến trang đăng nhập
4. **Hoặc:** Nhấn **"Đăng nhập bằng Google"** để sử dụng tài khoản Google

### 2.2 Đăng nhập
1. Nhập tên đăng nhập và mật khẩu
2. Nhấn **"Đăng nhập"**
3. Sau khi đăng nhập, hệ thống chuyển về Trang chủ

### 2.3 Học theo Chủ đề
1. Nhấn **"Chủ đề"** trên thanh menu
2. Chọn chủ đề phù hợp (lọc theo level: Beginner / Intermediate / Advanced)
3. Trang chi tiết chủ đề gồm:
   - **Từ vựng:** Danh sách từ vựng kèm phát âm, ví dụ, nghĩa tiếng Việt
   - **Bài học:** Nội dung bài học có hình ảnh, âm thanh, video
   - **Bài test:** Các bài kiểm tra Quiz, Listening, Reading
   - **Luyện nói:** Bài luyện phát âm với AI chấm điểm

### 2.4 Flashcard (Luyện từ vựng)
1. Từ trang chi tiết chủ đề → nhấn **"Flashcard"**
2. Hệ thống hiển thị thẻ từ vựng
3. Nhấn thẻ để lật xem nghĩa
4. Nhấn **"Đã nhớ"** hoặc **"Chưa nhớ"**
5. Sau khi hoàn thành, có thể **ôn lại từ chưa nhớ**

### 2.5 Làm bài kiểm tra
1. Nhấn **"Bài test"** trên menu → Chọn bài test
2. Hệ thống hiển thị câu hỏi trắc nghiệm
3. Chọn đáp án cho từng câu → Nhấn **"Nộp bài"**
4. Xem kết quả: Điểm, tỷ lệ đúng, chi tiết từng câu
5. Có thể **Làm lại** bài test

### 2.6 Luyện nói (Speaking)
1. Nhấn **"Luyện nói"** trên menu
2. Chọn bài speaking (yêu cầu tài khoản Pro)
3. Nhấn **"Ghi âm"** → Đọc theo bài mẫu
4. Nhấn **"Dừng ghi âm"** → Nhấn **"Chấm điểm"**
5. Hệ thống AI đánh giá: Phát âm, Lưu loát, Hoàn chỉnh, Điểm tổng

### 2.7 Ngữ pháp
1. Nhấn menu **"Thêm"** → **"Ngữ pháp"**
2. Chọn bài học ngữ pháp theo danh mục (Thì, Giới từ, Mệnh đề...)
3. Đọc nội dung bài giảng
4. Làm Quiz thực hành → Xem kết quả + giải thích

### 2.8 Tìm kiếm
1. Nhấn **"Tìm kiếm"** trên thanh menu
2. Nhập từ khóa (VD: "present", "food", "daily")
3. Hệ thống tìm kiếm trong: Chủ đề, Từ vựng, Bài test, Bài học, Ngữ pháp
4. Nhấn vào kết quả để truy cập nội dung

### 2.9 Dashboard (Theo dõi tiến độ)
1. Nhấn menu **"Thêm"** → **"Dashboard"**
2. Xem thống kê: Streak, XP, Level, Mục tiêu hôm nay
3. Biểu đồ: Điểm theo chủ đề, Phân bố kỹ năng
4. Tiến độ chi tiết theo từng chủ đề

### 2.10 Bảng xếp hạng
1. Nhấn menu **"Thêm"** → **"Xếp hạng"**
2. Xem top học viên có điểm cao nhất
3. Xem thứ hạng cá nhân của mình

### 2.11 AI Chatbot
1. Nhấn **icon robot** (góc dưới bên phải)
2. Nhập câu hỏi về tiếng Anh (ngữ pháp, từ vựng, dịch thuật...)
3. AI trả lời tức thì
4. Có gợi ý câu hỏi nhanh để bắt đầu

### 2.12 Nâng cấp Pro
1. Nhấn **"Nâng cấp Pro"** trong menu
2. Chọn gói: 1 tháng / 3 tháng / 1 năm
3. Thanh toán bằng:
   - **Chuyển khoản QR:** Quét mã QR → Chuyển khoản → Chờ duyệt
   - **Mã kích hoạt:** Nhập mã do admin cấp
4. Tài khoản Pro được kích hoạt sau khi xác nhận

### 2.13 Hồ sơ cá nhân
1. Nhấn tên user → **"Hồ sơ cá nhân"**
2. Xem/sửa Họ tên, Email
3. Đổi mật khẩu
4. Xem huy hiệu đã đạt và lịch sử bài test

## 3. Hướng dẫn cho Quản trị viên (Admin)

### 3.1 Truy cập Admin Panel
1. Đăng nhập bằng tài khoản admin
2. Nhấn **"Admin Panel"** trong menu dropdown

### 3.2 Dashboard Admin
- Xem tổng quan: Số user, đơn hàng, doanh thu, tăng trưởng
- User mới đăng ký gần đây

### 3.3 Quản lý người dùng
1. Nhấn **"Users"** trong admin nav
2. **Tìm kiếm:** Nhập tên/email để tìm user
3. **Phân quyền:** Thay đổi role (student/admin)
4. **Toggle Pro:** Bật/tắt trạng thái Pro cho user

### 3.4 Quản lý Chủ đề
1. Nhấn **"Chủ đề"** trong admin nav
2. **Thêm:** Nhấn "Thêm chủ đề mới" → Điền tên, mô tả, level
3. **Sửa/Xóa:** Nhấn nút tương ứng trên mỗi chủ đề

### 3.5 Quản lý Câu hỏi
1. Nhấn **"Câu hỏi"** trong admin nav
2. Chọn bài test → Xem/sửa câu hỏi
3. Thêm câu hỏi mới với đáp án A, B, C, D

### 3.6 Quản lý Mã kích hoạt
1. Nhấn **"Mã kích hoạt"** trong admin nav
2. Tạo mã mới: Chọn gói Pro → Nhấn **"Tạo mã"**
3. Gửi mã cho học viên để kích hoạt Pro

### 3.7 Quản lý Đơn nâng cấp
1. Nhấn **"Đơn nâng cấp"** trong admin nav
2. Xem danh sách đơn chờ duyệt
3. **Duyệt:** Nhấn ✓ để xác nhận thanh toán
4. **Từ chối:** Nhấn ✗ để từ chối đơn
5. Xem lịch sử đơn đã xử lý

### 3.8 Cài đặt hệ thống
1. Nhấn **"Cài đặt"** trong admin nav
2. Nhập **OpenAI API Key** để kích hoạt AI Chatbot + Speaking
3. Nhấn **"Test kết nối"** để kiểm tra

## 4. Tính năng hệ thống

### 4.1 Gamification (Trò chơi hóa)
| Tính năng | Mô tả |
|-----------|-------|
| **Streak** | Đếm số ngày học liên tiếp |
| **XP** | Điểm kinh nghiệm tích lũy qua hoạt động |

| **Badges** | Huy hiệu thành tích (Người mới, Bookworm, Speaker, Star...) |
